——格式问题

  1. 常用格式

    • 文章摘要

    使用<!-- more -->注释分隔摘要和正文

    • Front-matter

    Front-matter中也可以指定文章在要，以及其他内容自行指定最好填上title和date

  2. 公式、图表不显示

在Typora正常显示的公式图表在上传网页后，无法正常显示，Hexo 默认的 Markdown 渲染器不支持复杂公式，需要更换渲染器，详见Hexo Fluid配置指南。

  • 安装插件
  • 安装Pandoc

​ 具体教程可见https://www.cnblogs.com/Zhm17674730727/p/14030602.html

​ 命令行输入pandoc --version检查是否安装成功

  • 配置

在blog/themes/hexo-theme-fluid/_config.yml中修改配置，将math和mermaid中的enable和specific改为true，分别对应数学公式和流程图，specific是为了提升加载速度，之后书写博客时需要在文章的front-matter中指定math: true，mermaid: true

  • 其他
  1. 还有一些需要注意的地方，比如公式中的\符号，同时也是转义符，使用\\换行会出现无法显示的问题，因为渲染引擎会把\\渲染成\，再交给mathjax渲染公式，不修改配置情况下需要多加一个\写成\\\才能正常显示。
  2. 两个连续相同的花括号如{ {之间需要加空格，不然会报错
